# 🌶️ 고추밭 경매 시스템 v0.2.1

## 변경사항

- 기존 팀 수 2~10 제한 제거
- 팀 수 입력칸 제거
- `+ 팀 추가` 버튼으로 필요한 만큼 추가
- 각 팀별 `삭제` 버튼
- 최소 2팀 유지
- 팀 이름과 시작 포인트 개별 설정
- 설정 저장 시 경매 화면과 팀 현황 자동 반영

## 사용법

설정 → 팀별 설정 → `팀 추가`

필요한 만큼 팀을 만든 뒤 팀 이름과 포인트를 입력하고 `설정 저장`을 누릅니다.

> 기술적으로 고정된 최대 팀 수는 없습니다.  
> 다만 수십~수백 팀을 한 화면에 표시하면 화면과 브라우저가 느려질 수 있습니다.
