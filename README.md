# 🌶️ 고추밭 경매 시스템 v0.2.2

## 이번 수정

- 경매 화면 오른쪽 위에 실제 랜덤 룰렛 삽입
- 룰렛 결과가 현재 경매 선수로 즉시 자동 연결
- 별도 룰렛 메뉴에서도 추첨 결과 자동 연결
- 낙찰 또는 유찰 후 현재 선수를 비우고 다음 룰렛 대기
- 현재 경매 선수는 룰렛에서 중복 추첨되지 않음
- 팀별 영입 명단 표시
- 각 선수 이름, 티어, 라인, 낙찰가 표시
- 팀 카드 펼치기/접기
- 최근 낙찰 기록 유지
- 무제한 팀 추가 기능 유지

## 다음 단계

Supabase 연결 전 현재 기능을 직접 테스트합니다.

1. 오른쪽 룰렛 돌리기
2. 뽑힌 선수와 현재 경매 선수가 동일한지 확인
3. 팀 선택 후 입찰
4. 낙찰
5. 오른쪽 팀별 영입 명단과 낙찰가 확인
6. 낙찰된 선수가 다음 룰렛에서 제외되는지 확인
