# 🌶️ 고추밭 경매 시스템 v2.2.1

## 수정 내용

- 상단 `제3회 관동지방컵` 제목을 정중앙으로 재배치
- 제목 위 작은 영문 문구와 제목 사이 간격 정리
- 선수 카드 상단 티어 글자 확대
- 선수 카드 상단 주라인 / 부라인 글자 확대
- 선수 이름을 카드 정확한 중앙에 크게 배치
- 선수 카드 하단 기본 포인트 완전 제거
- `선택 완료` 배지 확대 및 금색 글로우 강화
- 낙찰 팀과 낙찰가 글자 확대
- 유찰 배지 강조

## 적용 방법

1. ZIP 압축 풀기
2. GitHub `kanto-cup-system`
3. `Add file` → `Upload files`
4. 압축 푼 폴더 안 파일 전부 업로드
5. `Commit changes`
6. Vercel 최신 배포가 `Ready`인지 확인
7. 사이트에서 `Ctrl + Shift + R`
