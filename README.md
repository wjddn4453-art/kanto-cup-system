# 🌶️ 고추밭 경매 시스템 v0.2.4

## 관전자 화면 완성 초안

- 현재 선수 / 티어 / 라인
- 현재 입찰가 / 최고 입찰팀 / 타이머
- 룰렛 진행 애니메이션 / 결과
- 남은 선수 수 / 전체 진행률
- 최근 낙찰
- 팀별 남은 포인트
- 팀별 영입 명단 / 선수별 낙찰가
- 낙찰 / 유찰 전체 화면 연출
- 운영 버튼 없는 보기 전용 화면

## 같은 컴퓨터 두 탭 동기화

Supabase 연결 전에도 같은 브라우저에서:

1. 첫 번째 탭: 실시간 경매
2. 두 번째 탭: 관전자 화면

으로 열면 BroadcastChannel과 localStorage를 이용해 룰렛, 입찰, 낙찰 연출을 동기화합니다.

다른 컴퓨터나 휴대폰까지 실시간 동기화하려면 다음 단계에서 Supabase를 연결해야 합니다.

## 테스트

1. 사이트를 두 탭으로 열기
2. 첫 번째 탭은 실시간 경매
3. 두 번째 탭은 관전자 화면
4. 운영자 탭에서 룰렛 돌리기
5. 관전자 탭에 룰렛 효과와 결과가 표시되는지 확인
6. 팀 선택 후 입찰
7. 관전자 탭에 가격과 최고 입찰팀이 표시되는지 확인
8. 낙찰
9. 관전자 탭에 낙찰 연출과 팀 명단이 표시되는지 확인
