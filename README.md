# 🌶️ 고추밭 경매 시스템 v1.2 — CARD DRAFT ARENA

사용자가 제공한 대회 경매 화면 구조를 참고해 경매 메인 화면을 카드 보드 방식으로 전면 개편했습니다.

## 핵심 흐름

1. 전체 선수 카드 확인
2. ALL / TOP / JUG / MID / ADC / SUP 필터
3. 룰렛 돌리기
4. 금색으로 강조된 선수 카드 클릭
5. 중앙 대형 선수 카드 화면
6. 선수를 영입할 팀 클릭
7. 낙찰가 입력 및 확정
8. 해당 팀 슬롯으로 선수 자동 배치
9. 전체 선수 카드에 선택 완료 표시

## 추가 기능

- 좌우 팀 현황 패널
- 팀별 5개 기본 슬롯
- 무제한 팀과 추가 선수 슬롯 지원
- 선수 사진 URL 지원
- 사진이 없으면 이름 이니셜 카드
- 선택 완료 선수 흑백 처리
- 유찰 표시 및 목록 연동
- 낙찰 연출
- 되돌리기

## 적용 방법

1. ZIP 압축 해제
2. GitHub `kanto-cup-system`
3. `Add file` → `Upload files`
4. 압축 푼 폴더 안 파일 전체 업로드
5. `Commit changes`
6. Vercel 최신 배포가 `Ready`인지 확인
7. 사이트에서 `Ctrl + Shift + R`

## 테스트

1. 경매 화면에서 룰렛 돌리기
2. 강조된 카드 클릭
3. 팀 카드 클릭
4. 낙찰가 입력 후 확정
5. 팀 슬롯과 포인트가 변경되는지 확인
6. 전체 카드에 선택 완료 표시 확인
7. 유찰 처리 확인
8. 라인별 필터 확인
