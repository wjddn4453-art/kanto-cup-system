# 🌶️ 고추밭 경매 시스템 v1.1 — PRO DRAFT UI

기존 기능은 유지하면서 화면을 프로 e스포츠 픽/밴 화면 감성으로 전면 개편한 버전입니다.

## 디자인 변경

- 어두운 청록색 프로 대회 그래픽
- 금색 포인트와 각진 프레임
- 중앙 선수 선택 카드 강화
- 팀 선택 버튼을 픽 슬롯 형태로 변경
- 입찰가와 타이머를 방송 그래픽처럼 강조
- 룰렛을 선수 선택 연출처럼 변경
- 낙찰 화면을 PICK LOCKED / SOLD 연출로 변경
- 관전자 및 프레젠테이션 화면 디자인 통일
- 모바일 반응형 유지

공식 게임 이미지, 로고, UI 자산을 복사하지 않고 프로 대회 픽창의 분위기만 참고해 새로 디자인했습니다.

## 적용 방법

1. ZIP 압축 풀기
2. GitHub `kanto-cup-system`
3. `Add file` → `Upload files`
4. 압축을 푼 폴더 안 파일 전체 업로드
5. `Commit changes`
6. Vercel 최신 배포가 `Ready`가 될 때까지 기다리기
7. 사이트에서 `Ctrl + Shift + R`
