# 고추밭 경매 시스템 v2.0

- 왼쪽 사이드바 제거
- 상단 경매 화면 / 선수 관리 / 설정만 유지
- 관전자 화면 제거
- 일반 룰렛과 유찰 전용 룰렛 분리
- 유찰 선수는 컴퓨터가 별도 기억
- 룰렛 이름이 세로로 흐르며 감속
- 선수 이름 중복 제거
- 티어 드롭다운 추가
- 부라인 `없음` 추가
- 둥글고 굵은 고딕 계열 UI
- 포켓몬 배틀·카드 분위기의 배경과 반짝임

## 적용
1. ZIP 압축 해제
2. GitHub 저장소에서 Add file → Upload files
3. 압축 푼 폴더 내부 파일 전체 업로드
4. Commit changes
5. Vercel Ready 확인
6. Ctrl + Shift + R

## 테스트
1. 일반 룰렛
2. 유찰 처리
3. 유찰 룰렛
4. 낙찰가 입력
5. 선수 관리 티어/라인 드롭다운
6. 설정 화면
